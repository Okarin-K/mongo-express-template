## About

MongoDB を使いたい時のバックエンドのテンプレートです。

個人的に使うサンプルです。

## 環境

-   OS: Ubuntu20.04
-   docker: 20.10.10
-   docker-compose: 1.29.2
-   TypeScript: 4.9.4
-   express: 4.18.2
-   mongodb: 4.13.0

## 使い方

install package.

```bash
npm ci
```

Starting docker container.

```
docker-compose up
```

Starting to server.

```bash
npm run start
```

access to ** http://localhost:3000/ **
